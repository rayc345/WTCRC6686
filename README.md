# WTCRC-A2Lithio
WTCRC DSP Radio control board for TEF665X and TEF668X V102/V205
By eggplant886

This project is built mainly based on the works of ace919(The UI) and pe5pvb(Lithio Driver).

The circuit is based on STM32F103 with LCD1602.
